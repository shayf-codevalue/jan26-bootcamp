# How to use this calculator
**This calculator had been created by Bahaa and Mariana**.

this calculator is working with python so you need to make sure you have it on your personal computer.
- python check: to check if you have python on your pc you need to run this code ``` python --virson ``` on your powershell
- if you see a python virson you can continue to the next bullet, if not you have to enter this prompt to download python ``` winget install Python.Python.3.12``` or just download from [python Download](https://www.python.org/downloads/)
- now you have to run the command ``` python .\calculator  operation number1 number2```
- you have 4 operation: 
```
1. add
2. subtract
3. multiply
4. divide
```
- all you need to do now is just choose operation and the numbers you need

**Have fun**
